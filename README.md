# ose-mcp

Welcome to the OSE (Old School Essentials) MCP System! This is a comprehensive solo RPG toolkit with 119 tools organized into 21 modules:
* Core Gameplay:
	* state (22 tools) - PC management, inventory, HP, conditions, clocks
	* mechanics (4 tools) - Dice rolls, reactions, morale, encounters
	* procedures (6 tools) - Turn/time tracking, dungeon exploration

* Character Development:
	* progression (6 tools) - XP, leveling up, class abilities
	* magic (6 tools) - Spellbooks, preparing & casting spells
	* encumbrance (3 tools) - Weight tracking, carrying capacity
	* light (4 tools) - Torch/lantern management

* World Building:
	* world (18 tools) - NPCs, factions, dungeons, hexes, towns, sites
	* stocking (2 tools) - Random room/hex content generation
	* tables (7 tools) - Custom encounter tables

* Gameplay Systems:
	* oracle (4 tools) - Yes/no questions, random events (for solo play)
	* adventures (4 tools) - Quest generation and tracking
	* encounters (3 tools) - Surprise, distance, pursuit/evasion
	* travel (3 tools) - Wilderness travel, weather, rations
	* hirelings (6 tools) - Recruiting, managing, and paying hirelings
	* economy (4 tools) - Buying/selling items, market prices

* Advanced Features:
	* consequences (8 tools) - Injuries, reputation, NPC memory
	* loot (1 tool) - Treasure generation
	* refs (3 tools) - Search OSE reference documents
	* gm (1 tool) - One-button automated GM procedures

* Utility:
	* ops (4 tools) - System diagnostics and help

To learn more about any module, try: ose_help_module("module_name") (e.g., "state", "world", "oracle")
